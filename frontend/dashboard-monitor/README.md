# dashboard-monitor (Angular)

Aplicación frontend del sistema de monitorización de sensores.

## Scripts

```bash
pnpm install
pnpm start
pnpm build
pnpm test
```

## Notas

- URL esperada del backend: `http://localhost:5024/api`
- Puerto de desarrollo frontend: `http://localhost:4200`
- La app usa Angular signals y bloques `@if` / `@else` / `@for`

Para documentación funcional completa, revisa `../README.md`.
